# Backend PHP

Esta pasta contém apenas o backend do projeto.

- `conexao.php`: ficheiro central de ligação com MySQL.
- `app/Core`: router, controller JSON e classe de base de dados.
- `app/Controllers/Api`: endpoints PHP usados pelo frontend.
- `app/Models`: modelos ligados ao MySQL.

O frontend não fica aqui. As páginas estão em `frontend/html`, os estilos em `frontend/css` e o JavaScript em `frontend/js`.
